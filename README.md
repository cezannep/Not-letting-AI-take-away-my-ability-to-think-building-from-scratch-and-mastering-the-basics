# Zance UI

A inspired futuristic animated UI experience built with React, TypeScript, Vite, Tailwind CSS, and Motion.

---

##  Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Motion
- ESLint

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/cezannep/Not-letting-AI-take-away-my-ability-to-think-building-from-scratch-and-mastering-the-basics.git
```

Navigate into the project:

```bash
cd "zance ui"
```

Install dependencies:

```bash
npm install
```

---

## 🛠️ Run Development Server

```bash
npm run dev
```

---

## 📦 Production Build

Build the project:

```bash
npm run build
```

---

## 🧹 Lint Project

```bash
npm run lint
```

---

## ⚡ Features

- Smooth animations using Motion
- Responsive futuristic UI
- Fast development with Vite
- Type-safe architecture with TypeScript
- Tailwind CSS styling system
- Clean scalable component structure
